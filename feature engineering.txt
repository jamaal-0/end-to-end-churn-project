the goal of feature engineering is to improve model input features, 
asking myself 'How can we modify or create features to help models learn better?'

Phase 1.
Handle the missing values by doing imputation.
if distribution is roughly symmetric use mean
if  data is skewed or outliers exist use median 


Phase 2.
Handle skewed features if the distribution heavily skewed or there is long tails present.
methods:
	•	Log transform
	•	PowerTransformer
	•	Yeo-Johnson

Phase 3.
Handle outliers if present by:
	•	Remove invalid rows
	•	Clip extreme values
	•	Use RobustScaler
	•	Apply transforms

Phase 4.
Encoding categorical features 

Phase 5.
Scaling numeric features.

Situation               Method
Normal numeric          StandardScaler
Outliers present        RobustScaler
Neural networks         MinMaxScaler
Sparse data             MaxAbsScaler

creating new features.

	•	Interaction features
